# Assets que precisam ser exportados do Figma

## Marca

- Logotipo Startup Hub — versão clara
- Logotipo Startup Hub — versão escura
- Símbolo Startup
- Selo Startup
- Favicon
- Fundo do rodapé

## Landing page

- Imagem principal do hero
- Textura/símbolo de fundo
- Imagem da seção de planos
- Elementos decorativos

## Formatos recomendados

- Logos e símbolos: SVG
- Fotografias: WebP, com PNG/JPG como fonte
- Favicon: SVG e PNG 32x32/192x192/512x512
