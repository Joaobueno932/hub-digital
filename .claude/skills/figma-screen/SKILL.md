# Implementação de tela pelo Figma

Referência: $ARGUMENTS

Utilize o Figma MCP para:

- identificar hierarquia;
- obter medidas e espaçamentos;
- consultar cores;
- consultar tipografia;
- identificar componentes reutilizáveis;
- identificar estados e variações;
- verificar imagens e ícones;
- compreender a navegação.

Antes de criar componentes novos, procure componentes equivalentes no projeto.

Não copie posicionamentos absolutos do Figma.

Converta o desenho para uma implementação:

- responsiva;
- acessível;
- semântica;
- reutilizável;
- compatível com o design system;
- adaptável a conteúdos reais.

Utilize os recursos existentes em public/assets.json.

Não recrie logos nem substitua a identidade visual.

Ao finalizar, compare visualmente:

- desktop;
- tablet;
- celular;
- comportamento com textos maiores;
- estados de erro;
- estados vazios;
- carregamento.
